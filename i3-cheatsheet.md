# i3wm Keybindings Cheatsheet

Config: `~/dotfiles/i3/.config/i3/config`

Modifier key:

- `$mod` = `Mod1` = `Alt`

## Applications and Launchers

| Keybinding | Action |
|---|---|
| `Alt + Enter` | Open terminal: `st` |
| `Alt + d` | Open Rofi app launcher |
| `Alt + Shift + d` | Open Rofi run launcher |
| `Alt + Shift + b` | Connect to Bluetooth headset |
| `Alt + Shift + F4` | Start camera stream |
| `Alt + Shift + F5` | Start desktop recording |
| `Alt + z` | Toggle screenkey |
| `Alt + Shift + a` | Show notification history |

## Window Focus

| Keybinding | Action |
|---|---|
| `Alt + h` | Focus left |
| `Alt + j` | Focus down |
| `Alt + k` | Focus up |
| `Alt + l` | Focus right |
| `Alt + Left` | Focus left |
| `Alt + Down` | Focus down |
| `Alt + Up` | Focus up |
| `Alt + Right` | Focus right |
| `Alt + Space` | Toggle focus between tiling and floating windows |
| `Alt + a` | Focus parent container |

## Resize Mode

| Keybinding | Action |
|---|---|
| `Alt + r` | Enter resize mode |

While in resize mode:

| Keybinding | Action |
|---|---|
| `h` | Shrink width |
| `j` | Grow height |
| `k` | Shrink height |
| `l` | Grow width |
| `Left` | Shrink width |
| `Down` | Grow height |
| `Up` | Shrink height |
| `Right` | Grow width |
| `Enter` | Exit resize mode |
| `Escape` | Exit resize mode |
| `Alt + r` | Exit resize mode |

## Audio

| Keybinding | Action |
|---|---|
| `XF86AudioRaiseVolume` | Increase volume by 5% |
| `XF86AudioLowerVolume` | Decrease volume by 5% |
| `XF86AudioMute` | Toggle output mute |
| `XF86AudioMicMute` | Toggle microphone mute |
| `F1` | Toggle output mute |
| `F2` | Decrease volume by 5% |
| `F3` | Increase volume by 5% |

## Screenshots

| Keybinding | Action |
|---|---|
| `Print` | Open Flameshot GUI |
| `Shift + Print` | Take full screenshot with Flameshot |
| `Alt + Print` | Open Flameshot GUI for current screen |

## i3 Management

| Keybinding | Action |
|---|---|
| `Alt + Shift + c` | Reload i3 config |
| `Alt + Shift + r` | Restart i3 in place |
| `Alt + Shift + Escape` | Exit i3 |

## Mouse
